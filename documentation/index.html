ColorCount Pro — Documentazione

Versione: 1.0   ·   Tipo: Single-file HTML5 web app   ·   Dipendenze: nessuna

1. Installazione

ColorCount Pro è una singola pagina HTML. Non serve compilare nulla.

  1. Estrai lo zip.
  2. Carica il contenuto della cartella docs/ sul tuo web server (o apri docs/index.html direttamente nel browser per upload di immagini).
  3. Per usare la fotocamera servono HTTPS e un permesso utente. localhost funziona anche senza certificato.
Hosting consigliato: GitHub Pages, Netlify, Vercel, Cloudflare Pages. Tutti gratuiti e con HTTPS automatico.

2. Uso base

Tab "Conteggio"

  • Avvia Camera → richiede permesso fotocamera, mostra il flusso live.
  • Carica → carica un'immagine dal dispositivo.
  • Scatta & Analizza → cattura il frame corrente e conta gli oggetti.
  • Rianalizza → ricalcola con nuovi parametri senza dover riscattare.

Funzioni supportate

sin, cos, tan, asin, acos, atan
log (base 10), ln (naturale), exp, sqrt, cbrt
abs, floor, ceil, round
Costanti: π, e
Memoria: Ans (ultimo risultato), M (se impostata)
Operatori: + - * / ^  ( )  !  %

Tasto SHIFT

Premi SHIFT per attivare la seconda funzione del tasto successivo (mostrata in arancione sopra il tasto principale). Esempio: SHIFT + sin = asin.

Esempi

G - Y                    → verdi meno gialli
(G + R) * 100 / (G+R+Y)  → percentuale verde+rosso sul totale
sqrt(G^2 + R^2)          → distanza euclidea
sin(π/4) * G             → mix matematico
3!                       → 6 (fattoriale)
50%                      → 0.5

4. Personalizzazione

Tutto il codice è in docs/index.html.

Aggiungere/modificare colori

Cerca const COLORS = [ e modifica l'array. Ogni colore ha:

{
  name: 'red',          // identificativo interno
  label: 'Rosso',       // etichetta UI
  var: 'R',             // variabile usata nella calcolatrice
  hex: '#e53935',       // colore del pallino + overlay
  ranges: [[0,15],[345,360]],  // intervalli Hue HSL (0-360)
  sMin: 0.35,           // saturazione minima
  lMin: 0.18, lMax: 0.82 // luminosità min/max
}

Cambiare tema

I colori dell'interfaccia sono nel blocco <style> in testa. Background: #0a0a0f, accent: #1e88e5.

Cambiare lingua

Le stringhe italiane sono inline nell'HTML e nei toast() del JS. Modifica direttamente.

5. Limiti noti

  • Il conteggio funziona meglio su sfondi neutri (bianco/nero/grigio).
  • Oggetti che si toccano vengono contati come uno (touching objects). Per separarli, alza la sensibilità o aumenta la distanza tra gli oggetti nella foto.
  • L'algoritmo è basato su HSL, non su deep learning — adatto a colori chiari, non a tonalità ambigue (es. cachi, beige).
